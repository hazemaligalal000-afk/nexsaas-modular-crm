## 🚀 DEPLOYMENT INSTRUCTIONS

### Local Development:
```bash
# 1. Clone repository
git clone https://github.com/your-org/nexsaas.git
cd nexsaas

# 2. Configure environment
cp .env.example .env
# Edit .env with your keys

# 3. Start services
docker compose up -d

# 4. Run migrations
docker compose exec php-fpm php modular_core/cli/migrate.php

# 5. Access application
# Frontend: http://localhost
# Backend API: http://localhost:8080
# AI Engine: http://localhost:8000
```

### Production Deployment:
```bash
# 1. Configure Kubernetes secrets
kubectl create secret generic nexsaas-secrets \
  --from-literal=db_password=xxx \
  --from-literal=stripe_secret_key=xxx \
  --from-literal=anthropic_api_key=xxx

# 2. Deploy application
kubectl apply -f k8s/configmap.yaml
kubectl apply -f k8s/deployment.yaml
kubectl apply -f k8s/service.yaml
kubectl apply -f k8s/ingress.yaml

# 3. Verify deployment
kubectl get pods
kubectl get services
kubectl get ingress

# 4. Check logs
kubectl logs -f deployment/nexsaas-backend
```
